# NooknKey

A restored 1890 Victorian short-term rental in The Nations, Nashville.

**Live site:** https://thenooknkey.com

## Files

- `index.html` — The full single-page website
- `vercel.json` — Vercel deployment configuration

## To update the site

Edit `index.html` directly on GitHub (click the pencil icon), commit, and Vercel auto-deploys within 30 seconds.
